# License Plate Detection and Recognition Using Machine Learning Models

## Master's Thesis
M.Sc. in Information Technology
Jahangirnagar University

## Overview
This project develops an automatic license plate detection and recognition system using machine learning and computer vision techniques.

## Technologies
- Python
- OpenCV
- TensorFlow
- EasyOCR
- YOLOV5

## Repository Structure
- src/ : source code
- dataset/ : sample dataset
- results/ : outputs and screenshots
- thesis/ : thesis document
